# 前台产品详情页布局需求文档（与后台产品新增表单1:1联动）

## 文档说明

1. **适用场景**：外贸 B 端独立站产品详情页，适配箱包类全品类产品，与后台「产品新增/编辑表单」完全数据联动
2. **核心目标**：后台完成表单填写与内容排版 → 前台自动按统一规范渲染完整页面
3. **交付标准**：可直接投喂 AI 生成 HTML/CSS/JS 前端代码，要求语义化结构、响应式适配、内置 GEO 优化规则
4. **数据映射规则**：页面所有可见内容、配置项 100% 对应后台产品表单字段，无硬编码内容

---

## 一、全局设计规范

### 1.1 页面尺寸与适配
- **PC 端**：主内容区最大宽度 1200px，水平居中
- **平板端**：768px-1199px，主内容区宽度 92% 自适应
- **移动端**：≤767px，单列全屏布局，左右留 16px 边距
- **页面整体**：纵向流式布局，所有模块按固定顺序从上到下排列

### 1.2 视觉风格规范
- **主色调**：青绿色 `#2ab39f`
- **辅助色**：深灰 `#333333`（正文）、中灰 `#666666`（次要文字）、浅灰 `#f5f5f5`（分隔背景）、白色 `#ffffff`（卡片背景）
- **字体**：`Arial, "Helvetica Neue", sans-serif`
- **间距规范**：模块间上下 40px，模块内上下 24px

### 1.3 技术基础规范
1. 全程使用 HTML5 语义化标签：`<header> <nav> <main> <section> <h1>-<h3> <footer>`
2. 所有图片默认懒加载，适配 WebP 格式
3. 页面加载核心指标：LCP≤2.5s，CLS≤0.1
4. 所有交互按钮、链接有明确 hover 状态

---

## 二、页面模块详细布局

### 模块 1：面包屑导航
- **数据来源**：后台「基础信息 - 产品分类」三级分类 + 产品简称
- **布局**：主内容区内左对齐，字号 14px，灰色文字
- **格式**：`Home > 一级分类 > 二级分类 > 三级分类 > 产品简称`
- **GEO**：自动生成 BreadcrumbList 结构化数据

### 模块 2：产品首屏核心区
- PC 端左右两栏（左 55% 主图 + 右 40% 信息），移动端上下布局

#### 左侧：主图轮播区
- 大图展示区高度 500px，图片居中自适应
- 下方横向缩略图列表 80px×80px，间距 10px
- 当前选中缩略图有 2px 青绿色边框
- **GEO**：每张图片 ALT = 品牌名 + 产品核心关键词 + 图片描述

#### 右侧：产品信息栏
1. **产品标题**：H1 标签，28px，加粗
2. **基础简介**：16px 灰色文字，行高 1.6
3. **Description 参数列表**：Type/Size/Color/Material/Features/Suit for
4. **询价操作区**：
   - 数量选择器（-/+/输入框）
   - `ADD TO INQUIRY` 主按钮（青绿色 #2ab39f）
   - 三个功能按钮：Send Email / Contact Now / Download As PDF

### 模块 3：产品规格参数（Product Parameters）
- 青绿色表头 `Product Parameters`，高度 50px
- 完整表格，左列参数名（25%，浅灰背景），右列参数内容（75%）
- 每行高度 40px
- **GEO**：H2 标签；参数注入 Product 结构化数据的 additionalProperty

### 模块 4：详情图文动态渲染区
- 根据后台区块类型数组循环渲染
- 支持区块类型：单图 / 多图网格(2/4宫格) / 标题(H2/H3) / 正文段落 / 左右图文
- 所有图片自动携带 ALT

### 模块 5：核心卖点（Quick Details）
- 青绿色表头 `Quick Details`，高度 50px
- 卖点列表：`加粗标题: 说明文字`
- 底部 Tags 标签行
- **GEO**：H2 标签；长尾关键词自然布局

### 模块 6：FAQ 问答
- H2 标题 `Frequently Asked Questions`
- 折叠手风琴样式，默认全部折叠
- **GEO**：无论是否展示，均生成 FAQPage 结构化数据

### 模块 7：关联产品推荐
- H2 标题（可配置，默认 `Product recommendation`）
- PC 端一行 4 张卡片，hover 上浮效果
- 移动端一行 2 个

### 模块 8：公共页脚
- 复用网站全局页脚

---

## 三、GEO 优化专属渲染规则
1. Meta 标签：title/description/keywords/canonical 全部来自后台配置
2. 结构化数据（3 类 JSON-LD）：Product / BreadcrumbList / FAQPage
3. 语义化标签：唯一 H1 = 产品标题，模块 H2 = 区块标题，详情内子标题 = H3

## 四、响应式适配
|断点|规则|
|---|---|
|≥1200px|左右两栏首屏、4 列推荐、4 宫格|
|768-1199px|首屏 5:4 左右、3 列推荐、2 行 2 列 4 宫格|
|≤767px|全部单列、首屏上下、2 列推荐、表格可滑动|

## 五、性能要求
1. 图片懒加载
2. 首屏关键 CSS 内联
3. 非核心 JS 异步加载
4. 返回顶部按钮常驻右下角
